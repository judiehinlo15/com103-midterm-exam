Step 1: Enter Basic Information
Input your name when prompted.
Enter your weekly budget.
If you type a non-numeric value (like letters), the program will ask you again until you enter a valid number.
Step 2: View Expense Categories

The program will display a list of categories:

Food
Transportation
School Supplies
Entertainment
Other
Step 3: Record Expenses (Up to 4 Entries)

You can log up to 4 expenses.

For each expense:

Enter a category number (1–5)
Enter 0 if you want to skip that entry
If you enter an invalid number, that entry will be skipped

If valid:

Enter a short description of the item
Enter the amount spent
If you enter a non-number, the program will ask again
Step 4: High Expense Alert
If an expense is more than 25% of your weekly budget, it will be marked with:
"! High Expense Alert!"
Step 5: View Summary

After all entries, the program will display:

Your name
Your weekly budget
A list of all recorded expenses:
Category
Description
Amount
Alert (if applicable)
Step 6: Budget Results

The program calculates:

Total Spent
Remaining Balance

Then shows your status:

If balance is 0 or more → “Budget OK! Keep it up.”
If balance is negative → “Overspent! Reduce spending.”
